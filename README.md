## STREETWEAR AFFORDABLE ##
A streetwear-themed mobile application developed using React Native and Expo. This app provides users
with a smooth and minimal interface to manage their account, track active orders, and view order history. It
is designed with a modern aesthetic inspired by contemporary streetwear brands, focusing on simplicity, bold typography, and structured layouts.

The application includes features such as account profile management, order status tracking, and a responsive UI optimized for both Android and iOS devices. Built for scalability, the project follows a component-based architecture, making it easy to expand with additional features like authentication, product browsing, and cart functionality.
